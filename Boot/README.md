# Install:
**Requirements:**
- Unlock Bootloader

**Instructions:** 
1. Install fastboot tools from google's sdk on your PC
2. Enter fastboot mode
3. Unlock your bootloader
4. Run "fastboot flash boot boot.img" in command line

**Note:** Replace boot.img, with the name of the kernel.
